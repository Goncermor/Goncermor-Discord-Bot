# Goncermor-Discord-Bot
### Discord bot written in Typescript with support for custom made plugins



## Functionalities
* Plugins (Add commands and functionalities)
* Lastest libraries and nodejs
* Easy to configure a new command
* Easy to create a plugin and add commands
* Pre made commands on example-plugins
